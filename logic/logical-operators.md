# XOR
**XOR** (aka **Exclusive or**) is a logical operator which is `true` if and only if the **inputs differ**.

<br>

Symbol: ⊕.

<br>

## The truth table of XOR
|A|B|A⊕B|
|:-|:-|:--|
|`0`|`0`|`0`|
|`1`|`1`|`0`|
|`0`|`1`|`1`|
|`1`|`0`|`1`|

<br>

## Two important properties of XOR
1. `n ⊕ n = 0`.
2. `n ⊕ 0 = n`.
